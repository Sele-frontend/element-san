# col
## usage:
```js
import elementSan from 'element-san';
const elCol = elementSan.elRow;

在父组件内:
    components:{
        'el-col':elCol
     }
```

## API

|  变量  | 可选值   | 说明 |
| ------ | ------ | -------- |
| span  | 0-24 | 栅格占据的列数	  |
| offset | 0-24 | 栅格左侧的间隔格数	 |
| push | 0-24 | 栅格向右移动格数	|
| pull | 0-24 | 栅格向左移动格数	|
